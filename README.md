# URL Search Killer

你是否苦恼过别人给你分享的链接[^1]，包含了一串又臭又长的 URL 参数[^2]？这款小工具[^3]就是来解决[^4]这个问题的！

只需要将 URL 丢进去[^5]，填写一下需要保留的参数（幸运的话可以自动填写）[^6]，然后就能访问一个干净的 URL！

## 使用方法

- 方法一：直接在网页中填写。
    - 输出 URL 后会出现一个跳转按钮，其本质上是一个链接而不是什么基于 JS 的假链接[^7]，也不会传递 Referer 标头[^8]，请放心使用它[^9]。
- 方法二：将 URL 拖放到网页中。
    - 如果规则命中，它会直接在新标签页打开[^10]。
- 方法三：通过 URL 参数 `url` 传入经过 URL 编码的 URL。（这是什么绕口令吗？）
    - 如果规则命中，它会直接重定向[^11]。

## FAQ

### 这款工具有什么特点？

几乎完美的无障碍支持[^12]，自动适配系统深色模式[^13]，简洁明了的界面设计[^14]，具有良好的可拓展性[^15]。

### 为什么要做一个这样的工具？

因为总有人给我发一串能盖满整个聊天窗口的链接[^16]。

### 会持续更新新规则吗？

只要我吃到了一坨新的，我就会来更新[^17]。

### 项目介绍是 AI 生成的吗？

我可不想让我的项目介绍里出现一堆莫名其妙的 emoji 符号[^18]。
