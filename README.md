# URL Search Killer

你是否苦恼过别人给你分享的链接，包含了一串又臭又长的 URL 参数？这款小工具就是来解决这个问题的！

只需要将 URL 丢进去，填写一下需要保留的参数（幸运的话可以自动填写），然后就能访问一个干净的 URL！

## 使用方法

- 方法一：直接在网页中填写。
    - 输出 URL 后会出现一个跳转按钮，其本质上是一个链接而不是什么基于 JS 的假链接[^1]，也不会传递 Referer 标头[^2]，请放心使用它[^3]。
- 方法二：将 URL 拖放到网页中。
    - 如果规则命中，它会直接在新标签页打开。
- 方法三：通过 URL 参数 `url` 传入经过 URL 编码的 URL。（这是什么绕口令吗？）
    - 如果规则命中，它会直接重定向。

## 贡献规则

内置规则中没有你需要的规则？请见 [CONTRIBUTING.md](CONTRIBUTING.md)。

[^1]: 尤其是那些使用了现代前端框架的网站，不规范使用链接的情形十分常见。使用 JS 实现链接跳转不仅会破坏浏览器默认行为，还会严重阻碍无障碍辅助技术准确识别链接。
[^2]: 某些网站会根据 Referer 标头阻止你访问他们不愿意公开分享但又没有理由删除的内容。
[^3]: 这里是指这个按钮会按照预期的行为运行，完全没有 JS 脚本偷偷给链接塞追踪参数，也不会进入一个页面告诉你即将离开网站请注意安全。所以请你自行判断你要访问的链接是否安全，但你都用上这个工具了想必你应该知道你在干什么。
